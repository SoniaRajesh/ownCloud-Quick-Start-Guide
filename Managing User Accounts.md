
# Managing User Accounts
Administrators can add, edit, and delete user accounts through ownCloud web interface. 
![enter image description here](https://doc.owncloud.org/server/9.1/admin_manual/_images/users-config.png) 

## Adding a user account

 1. In the **Username** field, type a user name for the new users (for example, BeckyWillioms).
    >The username can include letters (a-z, A-Z), numbers (0-9), dashes (-), underscores (_), periods (.), and at signs (@).
    
 2. In the **Password** field, type a password.
 3. In the **Group** drop-down list, select an appropriate group.
 4. Click **Create**. The system displays a message indicating the successful creation of a new user account.
